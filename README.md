# Signup Page UI

A simple and responsive Signup Page UI built using **HTML, CSS, and JavaScript**.

## Features

- Responsive signup form design
- Floating input labels
- Password show/hide toggle
- Password validation
- Confirm password checking
- Clean modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
